# Fee Matrix

## Fee Types
- Referral Fee
- Placement Fee
- Sourcing Fee
- Coordination Fee
- Relationship Management Fee
- Network Access Fee
- Pass-Through Fee

## Mode Mapping

### Referral Mode
- flat referral fee
- % of first invoice

### Placement Mode
- per-worker placement fee
- optional flat coordination fee

### Retained Relationship Mode
- recurring relationship fee
- optional sourcing / placement fee

### Contractor Mode
- labor invoice
- coordination fee
- pass-throughs
- margin

## Protection Escalation

### Stage 2
